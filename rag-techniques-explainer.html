💡 Why this matters

RAG systems power modern AI assistants and search engines by combining retrieval accuracy with language model reasoning. These techniques help AI systems access up-to-date information and domain-specific knowledge without retraining models.

🔑 Key concepts

RAG (Retrieval-Augmented Generation)
A technique that combines document retrieval with language models to provide accurate, context-aware responses.
Dense Embeddings
High-dimensional vector representations where semantically similar texts have similar vector values.
FAISS
Facebook's library for efficient similarity search and clustering of dense vectors.
BM25
A classic information retrieval algorithm that ranks documents based on term frequency and document length.
HyDE
Hypothetical Document Embeddings that generate a fake answer and use its embedding for retrieval.
Cross-Encoder
A neural network that takes both query and document as input to compute their relevance score.
