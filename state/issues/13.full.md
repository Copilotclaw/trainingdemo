### Comment 3 — github-actions (2026-03-16T11:31:01Z)

Google Colab works **notebook by notebook** — there's no way to open an entire repo as a "project" in one click.

**Your options:**

1. **Open single notebooks via URL** — use this pattern:
   ```
   https://colab.research.google.com/github/schuerstedt/ragmcp/blob/main/<path/to/notebook.ipynb>
   ```

2. **Clone the full repo inside a Colab notebook** — at the top of any notebook, add a cell:
   ```python
   !git clone https://github.com/schuerstedt/ragmcp
   %cd ragmcp
   ```
   This gives you all files, scripts, workers, etc. available for that session.

3. **"Open in Colab" badges** — I can add badges to the `ragmcp/` notebooks' READMEs pointing directly to each notebook on Colab, if that helps navigation.

**Recommendation**: Option 2 is best if the notebooks depend on each other or shared helper files — clone the whole repo at notebook start.

Want me to add a `!git clone` cell to the ragmcp notebooks, or add Colab badge links to the README?
