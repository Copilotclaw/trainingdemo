### Comment 1 — github-actions (2026-03-16T15:08:15Z)

Now let me build the notebook:

✅ **`rag-techniques-ollama.ipynb`** committed to main.

The notebook covers all six techniques in order:

1. **Dense Embedding** — FAISS + `all-MiniLM-L6-v2`, semantic search
2. **BM25** — `rank-bm25`, keyword/sparse retrieval
3. **HyDE** — LLM generates a hypothetical answer, embed that instead of the query
4. **MultiQuery** — LLM expands query into variants, retrieves for each, merges
5. **Step-Back** — LLM abstracts the query upward, retrieves broader context
6. **Re-Ranking** — cross-encoder (`ms-marco-MiniLM-L-6-v2`) does a final accurate pass

Plus a **full pipeline cell** that chains all six together into one `advanced_rag()` function.

**Deps**: `sentence-transformers faiss-cpu rank-bm25 ollama` — `!pip install` cell is at the top. Works locally or on Colab.
