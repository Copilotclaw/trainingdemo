🔑 Key concepts

RAG
Retrieval-Augmented Generation - combining document search with large language models
Embeddings
Numerical representations of text that capture semantic meaning in vector space
FAISS
Facebook AI Similarity Search library for efficient vector similarity operations
Semantic Search
Finding documents based on meaning rather than exact keyword matches
